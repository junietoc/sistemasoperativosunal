Los sistemas operativos (OS) son programas que actúan de intermediarios entre el hardware del PC y el usuario, ayudando a manejar
de manera más cómoda cada una de las aplicaciones del PC. Los OS tienen privilegios especiales que les permiten manejar a otros programas.
Funciones Principales:
- Administrar hardware para realizar tareas: Cuando una aplicación comienza a correr, por ejemplo Spotify, el sistema
operativo determina qué y cuánta memoria va a designar para que Spotify corra sin problemas.
- Organizar datos para el fácil acceso: Cuando los archivos almacenados en la memoria del PC necesitan ser accedidos,
el sistema operativo se encarga de organizarlos para que su acceso sea rápido y eficiente.
- Comunicación con periféricos: El sistema operativo está preparado para comunicar información entre las aplicaciones y
los periféricos, por ejemplo comunica Microsoft Word con el teclado para que la aplicación funcione correctamente.


Características de los OS:
- Están construidos para evolucionar, ir mejorando versión a versión y corregir errores de software de versiones pasadas.
- Facilitan la experiencia de usuario con interfaces de usuario que hacen el manejo del PC fácil e intuitivo.

Tipos de OS:
- Multitarea: Son capaces de ejecutar múltiples aplicaciones y múltiples funciones al mismo tiempo.
- Monousuario: Permiten el manejo del PC por un usuario que tiene acceso a todos los recursos del mismo
- Multiusuario: Están construidos para compartir los mismos recursos del PC en varios usuarios conectados al mismo.

HISTORIA DE LOS OS:
Los sistemas operativos nacen de la necesidad de ejecutar programas en los computadores de maneras eficientes. En los 50s los
computadores se estaban volviendo cada vez más rápidos, el problema estaba en que los programas eran cargados a los computadores
por personas, y esto tardaba un tiempo. Así, el primer OS empezó siendo un programa que cargaba automáticamente programas.
Pero el anterior problema no es el único que los OS solucionaron. Algunos programas se comunicaban con periféricos como impresoras,
pero por cada impresora existía un protocolo, así los programadores necesitaban escribir un programa por cada periférico. Así que los OS
se encargaron de ser los intermediarios entre los programas y los periféricos.
Como última característica, los OS permitieron una nueva posibilidad: ejecutar varios programas al mismo tiempo.

1980: QDOS por Tim Patterson.

1983: LisaOS por Apple.
